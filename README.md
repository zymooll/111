# 长沙高校餐饮信息档案

本仓库归档长沙多所高校校内食堂档口及周边餐饮商家的公开资料。

## 文件说明

- `data/merchants.csv`：商家、位置、历史价格、评价摘要与可信度。
- `data/images.csv`：已归档图片、对应商家、原始 URL 与版权说明。
- `data/csut_merchants.csv`：长沙理工大学金盆岭、云塘两校区餐饮数据。
- `data/csut_images.csv`：长沙理工大学两校区图片索引。
- `data/hunan_universities_merchants.csv`：湖南农大、湖南工商、中南、湖大、湖师大餐饮数据。
- `data/hunan_universities_images.csv`：上述五校的图片索引。
- `docs/research-report.md`：中文调研结论与使用注意事项。
- `docs/sources.md`：来源清单、采集方法与限制。
- `docs/csut-research-report.md`、`docs/csut-sources.md`：长沙理工大学专项报告与来源。
- `docs/hunan-universities-research-report.md`、`docs/hunan-universities-sources.md`：五校专项报告与来源。
- `assets/merchant-images/`：公开文章中的代表性图片。
- `assets/csut-images/`：长沙理工大学两校区代表性图片。
- `assets/hunan-universities-images/`：五校食堂、菜品与周边生活圈代表性图片。

## 重要限制

资料来自不同年份的高校官网、校级媒体和公开攻略，并在 2026-07-18 采集。商家可能已迁址、改名、调价或停业，不能直接视为 2026 年实时营业清单。所有图片仅作资料索引和研究归档，版权归原作者或平台所有，许可证未知；对外使用前应再次取得授权。
