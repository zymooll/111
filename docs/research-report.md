# 中南林业科技大学食堂与周边餐饮调研（首轮归档）

## 结果概览

本轮归档形成 42 条餐饮记录：6 个食堂级地点、14 个校内档口/推荐点位、22 个校外商家或摊位；另归档 19 条图片索引和 19 个本地图片文件。

可核验度最高的是 2019 年校内食堂指南和 2018 年周边美食地图。两篇文章都能读取完整正文，因此店名、当时价格、推荐菜和体验摘要可以追溯。最大问题是时间：距本次采集已有 7—8 年，所有营业状态都应视为“待实地核验”。

## 校内食堂结构

- 林海食堂：两层，靠近主教和东园足球场，人流大。文中可确认“蒸蒸日上”“拌的码粉面馆”。
- 林语食堂：以明亮、干净和温馨著称，二楼曾可通往后街和多福超市。可确认“重庆小面”“香来客”。
- 林苑食堂：规模较大、两层、有空调、靠窗座位适合自习。可确认“清香荷叶饭”“河南肉夹馍”。
- 林涛食堂：近西园宿舍区和球场。可确认“千里香馄饨”“鸡扒饭”。
- 林冠食堂：较安静、饭点相对不拥挤。可确认“小明家”“福州煨汤”。
- 林旺食堂：只在位置描述中出现，缺少独立资料。

历史价格集中在 6—12 元/份；这只能反映 2019 年学生餐饮价格带，不能作为当前预算。

## 周边餐饮带

### 科大路后街

文章把科大路与青园路交叉区域描述为主要美食聚集区。强推荐包括螺鼎记螺蛳粉、台湾脆皮玉米、祖师爷土家掉渣饼、杜四喜烤冷面、河南肉夹馍和台北101。继续过西门向北还能看到北京烤鸭、统桶发卤粉王和泡菜点吧。

### 青园路

代表商家包括香酥鸡蛋灌饼、邓哥麻辣烫、和财记油炸社、一芳、杨国福麻辣烫、津市牛肉米粉、小肥花甲粉和新西兰·岩烧乳酪。邓哥麻辣烫的 16:30 开门、共享圆桌自煮等信息很有辨识度，但都是 2018 年描述。

### 东门与韶山南路方向

来源以榜单形式提到津市刘聋子粉馆、刘大姐酸萝卜、鸭子粉、一米香盖码饭和辉胖子，细节较少，因此可信度为中或低。

## 评价摘要中的高频信号

- 性价比：校内档口多用“物美价廉、分量足、出饭快”描述。
- 口味：螺蛳粉强调酸笋、小料和辣度；邓哥麻辣烫强调卤味；和财记强调现炸肉类。
- 环境：林语偏“明亮温馨”，林苑偏“空间大、可自习”，林冠偏“安静、不拥挤”。
- 时段：千里香馄饨、河南肉夹馍和螺蛳粉曾被描述为晚间仍可购买；当前时段需重查。
- 风险：等待时间、偏油、辣度高和开门时间不稳定，是历史评价中最明确的负面信息。

## 下一轮核验建议

1. 实地或电话确认 42 条记录的“营业/迁址/改名/停业”。
2. 记录 2026 年菜单、价格、营业时间和支付方式。
3. 每家至少采集一张自有版权的门头照和一张菜品照，替换许可证未知的历史图片。
4. 对评价采用结构化问卷：口味、价格、分量、卫生、服务、排队时间各 1—5 分，并保存样本量而非个人身份。
