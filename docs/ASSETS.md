# 演示素材

首版菜品封面使用项目内的原创 SVG 占位插画，位于 `apps/user-web/public/dishes/`。这些文件不依赖第三方图片站，避免外链失效、跨域、授权和离线缓存问题。

上线前可由管理端上传正式菜品照片并替换演示数据中的 `image_url`。若使用第三方素材，应在本文件补充作者、原始页面、许可证和获取日期。

## 远端历史图片

`assets/merchant-images/` 与 `data/images.csv` 是仓库原有的历史调研档案。19 张图片来自公开文章或搜狐 CDN，许可证均记录为“未知”，部分图片含学校、学生组织或平台水印、品牌文字和二维码。它们仅用于来源定位与内部核验，不属于本项目可公开再发布的占位素材。

因此不得把这些图片复制到 `apps/user-web/public/`、通过 API `/media` 暴露、热链原始 URL，或用于训练与正式展示。取得作者或权利人许可后，方可在本文件登记授权范围并迁移到公开素材目录。
